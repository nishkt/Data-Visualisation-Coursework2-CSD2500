Common Data Visualisation Software

Desktop Software

Tableau

Tableau helps users create and share data insights across your organisation. Through the use of various graph types and data organisers, users will be able to present data on many different platforms through the Tableau Server. For more information access the Tableau webpage here!

TIBCO Spotfire

TIBCO Spotfire is another software that helps create and share data visualisation techniques with numerous users. Users could be based at home, professional atmospheres, or non-profit organisations. For more information on the specifics of TIBCO Spotfire, click here

qlik

Qlik lets users create visualisations and makes the data in a company more understandable. For more information, click here

Development Library

D3JS

D3JS, also know as D3.js or Data-Driven Documents js, is a JavaScript library used to organise documents based on data that the user provides. Through the use of HTML, SVG, and CSS, D3JS is able bring data to life in a comprehensible and coherent way. For more information on D3JS, click here

Vega (Github)

Vega, brought to you by github, is a library used to create, save, and share data visualisation designs in the output form of HTML5 Canvas and SVG. For more information on Vege, click here

Processing

Processing is a software and laguage for learning how to code under the context of visual arts. Processing is a software that integrates the uses of visual data with programming to maximize the understanding of data. For more information on Processing, click here
