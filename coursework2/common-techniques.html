Common Technique in Data Visualisation

1D/Linear

This type of representation of data is one dimensional. It is commonly not visualised and is organised by a single feature. A common example of 1D/linear data visualisation is a list of items.

2D/Planar

This type of data visualisation mostly revolves around geospaces and information represented on maps. Below are some examples of 2D/Planar data visualisations

Cartogram

Dot Distribution Map

3D/Volumetric

3D/Volumetric are data represented in three dimensional spaces. This data representation involves three different components (such as the x, y, and z components).

Temporal

Temporal data representation is data shown in a specific time period. As time goes on, variances and fluctations of data is shown. Examples include the following:

Timeline

Gantt Chart

nD/Multidimensional

This the most common type of representation of data that people use. Numerous dimensions are represented in the visualisation. Examples include the followiing:

Bar Chart

Histogram

Pie Chart

Tree/Hierarchical

This representation is commonly used in ordering tasks or people in a certain rankings and representing that data visually. Examples of Hierarchical data visualisation includes:

General Tree Visualisation

Radial Tree

Network

Network data visualisation shows relationships between numerous different components to end up creating a network. Often looks like a very complex spider web. Examples include:

Matrix

Dependancy Graph

Subway/tube map

Programming/Code

Code can be used to create visualisation of data. Below is example of some code in HTML used to create a table

<caption>Quarterly Financials for 1962-1964 (in Thousands)</caption> <tr> <th scope="col">1962</th> <th scope="col">1963</th> <th scope="col">1964</th> </tr> <tr> <td>$145</td> <td>$167</td> <td>$161</td> </tr> <tr> <td>$140</td> <td>$159</td> <td>$164</td> </tr> <tr> <td>$153</td> <td>$162</td> <td>$168</td> </tr> </table>

which will create this:

Quarterly Financials for 1962-1964 (in Thousands)
1962 1963 1964
$145 $167 $161
